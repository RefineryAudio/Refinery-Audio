Refinery Audio – Prestige Build

Refinery Audio is a simple audio file cleaner and tag editor designed to help you quickly organize and rename your MP3 files. It supports manual edits, batch renaming, and metadata management.

Features:

  • Add folders or individual MP3 files
  • Preview and clean up filenames
  • Edit metadata: Title, Artist, Album, Year, Genre
  • Undo changes if needed
  • Nuke metadata on all audio files (optional)
  • Prestige-enabled row colors for easier navigation

Installation:

  1. Extract the ZIP folder anywhere you like.
  2. Run Refinery_Audio.exe – no installation required.

Usage:

  1. Click "Add Folder" or "Add Files" to load your MP3s.
  2. Edit artist or titles manually, or let Refinery Audio clean them automatically.
  3. Use "Apply Changes" to rename files and update metadata.
  4. Optionally, use "Export Manifest" to save a text report of your session.
  5. Press "Undo" if you need to revert recent changes.
