# Refinery Audio

**Refinery Audio** is a free desktop tool for cleaning, organizing, and refining MP3 files.  
Quickly rename tracks, edit metadata, and keep your audio library tidy.

---

## Features

- Automatic renaming: `Artist – Title` or `Title Only` modes  
- Manual renaming and metadata editing for individual tracks  
- Metadata fields supported: Title, Contributing Artist, Album Artist, Album, Year, Genre  
- “Nuke Metadata” option to completely remove ID3 tags  
- Undo support for accidental edits  
- Preview new file names before applying changes  
- Export a Change Manifest to log all edits  
- Dark-themed UI with customizable colors

## How to Use

1. Download the latest release from the [Releases page](https://github.com/YOURUSERNAME/Refinery-Audio/releases).  
2. Add your MP3 files or folders using the toolbar.  
3. Adjust naming and metadata options.  
4. Click **Apply Changes** to rename files and update metadata.  
5. Optionally, export a **Change Manifest** for record-keeping.
